# Antigravity Skills

Este repositorio contiene las habilidades (Skills) personalizadas para el entorno Antigravity AI.

## Skills Incluidas:
- **Brainstorming**: Proceso socrático para diseño técnico.
- **Planning**: Descomposición de tareas para implementación.
- **Reddit Posts Scraper**: Extracción de datos de Reddit usando Playwright.
- **Skill Architect**: Creación de nuevas habilidades modulares.
- **Troubleshooting**: Diagnóstico y resolución de errores.
